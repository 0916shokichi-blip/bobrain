# インディーハッカーのコツ

個人開発で収益を出すためのよくある原則。

## 外れくじを早く引く

1 つのプロダクトに全部を賭けるより、小さく複数試して早く外す。Claude Code のような AI 支援ツールで開発コストが劇的に下がった今、試行回数が武器になる。

## Build in Public

開発過程を X や Bluesky で公開する。フォロワーが付いてローンチ時の初速になる。数字とスクショと学びをセットで投稿するのが定石。

## LP × Stripe の最小マネタイズ

プロダクトを本格実装する前に、LP と Stripe リンクだけで需要検証するのが最小構成。
